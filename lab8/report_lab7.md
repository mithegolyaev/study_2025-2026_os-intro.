## 2. Теоретические сведения

### 2.1 Файловая система Linux
Файловая система Linux представляет собой иерархическую структуру каталогов, 
начинающуюся с корневого каталога `/`.

Основные типы файловых систем:
- ext2
- ext3
- ext4
- xfs
- ntfs
- fat

### 2.2 Основные команды

#### Создание файлов
```bash
touch file.txt
```

#### Просмотр файлов
```bash
cat file.txt
less file.txt
head file.txt
tail file.txt
```

#### Копирование
```bash
cp file1 file2
cp -r dir1 dir2
```

#### Перемещение
```bash
mv file1 file2
```

#### Права доступа
```bash
chmod u+x file
chmod 755 file
```

## 4. Права доступа

Права доступа делятся на:
- владелец
- группа
- остальные

Пример:
```
-rwxr--r--
```

Цифровое представление:
- 7 = rwx
- 6 = rw-
- 5 = r-x
