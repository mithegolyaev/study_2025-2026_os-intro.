# Потоки ввода-вывода

В Linux существуют 3 стандартных потока:

- **stdin (0)** — ввод (клавиатура)  
- **stdout (1)** — вывод (консоль)  
- **stderr (2)** — ошибки  

Программы по умолчанию используют эти потоки.

---

# Перенаправление ввода-вывода

Основные операторы:

- `>` — перезапись файла  
- `>>` — добавление в файл  
- `<` — ввод из файла  

Примеры:

ls > file.txt
ls >> file.txt


Также:

2> error.txt
&> all.txt


---

# Конвейеры (pipe)

Конвейер передает вывод одной команды в другую:


команда1 | команда2


Пример:

ls -l | sort


Используется для обработки данных и автоматизации.

---

# Команда find

Используется для поиска файлов:


find путь -name "шаблон"


Примеры:

find ~ -name "f*"
find /etc -name "p*"


Удаление найденных файлов:

find ~ -name "*~" -exec rm {} ;


---

# Команда grep

Используется для поиска текста:


grep строка файл


Примеры:

grep begin file.txt
ls -l | grep лаб


Позволяет фильтровать вывод других команд.

---

# Проверка использования диска

Команды:

- **df** — показывает разделы диска  
- **du** — показывает размер файлов  

Примеры:

df -vi
du -a ~/


Используются для анализа памяти.

---

# Управление процессами

Основные команды:

- `ps aux` — список процессов  
- `kill PID` — завершение процесса  
- `jobs` — список задач  

Фоновый запуск:

gedit &


Процесс имеет уникальный PID.
